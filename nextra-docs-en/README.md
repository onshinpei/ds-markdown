# ds-markdown English Documentation

This is the English version of the ds-markdown documentation, built with Nextra.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `out/` directory.

## Deployment

This project is automatically deployed to GitHub Pages via GitHub Actions. The built files are deployed to the `gh-pages` branch under the `/en` directory.
