# API Reference

ds-markdown provides two main components: `DsMarkdown` and `MarkdownCMD`.

## Component Overview

### DsMarkdown

A basic Markdown typing animation component, suitable for static content.

```tsx
import DsMarkdown from 'ds-markdown';

<DsMarkdown interval={20} answerType="answer">
  # Content
</DsMarkdown>
```

### MarkdownCMD

A Markdown component that supports streaming data, suitable for handling real-time data streams.

```tsx
import { MarkdownCMD } from 'ds-markdown';

const ref = useRef(null);
ref.current?.push('New content');
```
