# API 文档

ds-markdown 提供了两个主要组件：`DsMarkdown` 和 `MarkdownCMD`。

## 组件概览

### DsMarkdown

基础的 Markdown 打字动画组件，适合静态内容。

```tsx
import DsMarkdown from 'ds-markdown';

<DsMarkdown interval={20} answerType="answer">
  # 内容
</DsMarkdown>
```

### MarkdownCMD

支持流式数据的 Markdown 组件，适合处理实时数据流。

```tsx
import { MarkdownCMD } from 'ds-markdown';

const ref = useRef(null);
ref.current?.push('新内容');
```
